# 引用规范

## 1. 飞书知识库引用（飞书文档跳转）
- **知识库跳转**：在Markdown中直接嵌入飞书知识库链接，格式为：
  
  ```markdown
  [《文档标题》-知识库](飞书知识库永久链接)
  ```
  
- **文档跳转**：在Markdown中直接嵌入飞书文档链接，格式为：
  
  ```markdown
  [《文档标题》](飞书文档永久链接)
  ```
  
- **锚点跳转**：需跳转到文档内具体章节时，可使用飞书文档的「锚点链接」，格式为：
  ```markdown
  [《文档标题》#章节锚点](飞书文档永久链接#heading=hxxxxxx)
  ```

## 2. 外部资源引用
- **外部文档/文章**：需标注来源平台，格式为：
  
  ```markdown
  [《资源标题》](外部链接) - 来源：平台名称（如CSDN/知乎/官方文档）
  ```
  
- **代码仓库引用**：链接到GitHub/Gitee仓库时，需明确仓库名称与用途：
  ```markdown
  代码仓库：[MTI机器人队四足仓库](https://github.com/Lain-Ego0/ROBOCON-BRS_robot) - 用途：存放强化学习步态训练代码
  ```

## 3. 附件与资源引用
- 图片/工程文件等附件需上传至飞书文档的「附件库」，并在文档中标注附件说明：
  ```markdown
  参考装配图：![四足机器人腿部装配图](飞书附件链接)
  ```
- 大型工程文件需上传至飞书文件夹，并在文档中链接到文件夹：
  ```markdown
  机械模型文件：[四足机器人腿部结构模型](飞书文件夹链接)
  ```

## 4. 版本与权限说明
- 飞书文档默认开启「版本历史」功能，重要文档更新需在「更新日志」中记录版本变化；
- 知识库按模块设置访问权限：核心技术文档仅对队内成员开放，公开教程类文档可设置为「互联网可见」，此仓库作为Public。
